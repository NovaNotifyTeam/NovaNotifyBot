NovaNotify Discord Bot

Setup:
1. Install Node.js
2. Open folder in terminal
3. Run: `npm install`
4. Rename `.env.example` to `.env`
5. Fill in `.env` file
6. Run: `npm start`

Command:
/announcmentmemes - announcement basic embedded message custom color and Title for message
/dmuser - dms a specific user from the bot (unless they have dms off)
/dmannounce - dms a announcement to every member besides the ones that have dms off
